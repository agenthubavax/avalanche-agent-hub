## 1. Product direction

Avalanche Agent Hub should prioritize one sharp wedge:

Open-source Avalanche-native toolkit for programmable on-chain quests and agent operations.

This is stronger than trying to position as a broad AI-agent super-platform.

## 4. Competition summary

| Bucket | Examples | Why they matter |
|---|---|---|
| Agent SDKs | Coinbase AgentKit, GOAT SDK, Solana Agent Kit, ElizaOS | Compete for builder mindshare on agent execution + on-chain actions |
| Quest / growth platforms | Galxe, Layer3, RabbitHole, Zealy, TaskOn | Compete if Agent Hub is framed as a quest marketplace |
| Wallet / app infra | Crossmint, thirdweb | Compete if Agent Hub is framed as generic wallet/app infra |

Strategic takeaway:
Do not compete on pure audience/distribution first.
Do not compete on generic agent breadth first.
Compete on embedded programmable infra + Avalanche-native depth + open-source trust.
